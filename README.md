# TrustID

TrustID is a privacy-focused identity vault and smart form assistance platform designed to reduce repetitive data entry and human error.

## Problem Statement
Students and users repeatedly fill Google Forms, KYC forms, registrations, and applications containing the same information. This process is repetitive, time-consuming, and prone to errors.

## Solution
TrustID securely stores user information and assists in intelligent form filling while keeping users in control of their sensitive data.

## Features

### Vault
Stores user information securely for future use.

### Smart Form Assistance
Helps autofill repetitive fields while allowing users to choose whether sensitive information should be inserted automatically or require confirmation.

### Audit / History
Maintains records of forms used, edits, and activity history.

### Privacy First Architecture
TrustID is designed so that the platform creators themselves cannot access user personal information. Data ownership remains with the user, and only the user has access to their stored information.

## Benefits
- Reduces repetitive form filling
- Minimizes human error
- Protects user privacy
- Gives user-controlled autofill
- Maintains activity history

## Future Scope

- Google Form integration
